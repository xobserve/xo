对于企业应用而言，能够定制侧边导航栏是非常常见的需求，Datav 允许我们在每个团队中轻松创建自己的侧边导航栏，用户可以在这些侧边导航栏之间切换。


<img src={require('@/img/docs/admin/sidebar.jpg').default.src} alt="global-roles"  className="rounded-lg ring-1 ring-black/5"  style={{height: '500px'}}/>


在上面图表中，红色箭头所指的区域是可以被定制化的，而下方的区域则是留给了系统必须的一些图标，无法被定制。

在之前章节中，我们已经介绍过侧边导航栏的相关概念，如果大家还不了解，可以先阅读 [侧边导航栏](/docs/tutorial-sidebar) 一章。
