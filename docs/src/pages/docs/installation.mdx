## Precompiled binaries
### <Heading number="1">Download from Github</Heading>
Visit <a href="https://github.com/apm-ai/datav/releases" target="_blank">Github</a> to download the latest release.



### <Heading number="2">Running datav</Heading>

```bash
## Switch to the directory datav resides
cd YOUR_DATAV_PATH

### Start datav
./datav
```

The service is now started, visit http://localhost:3003 and login. Welcome to the datav platform.

## Running in development mode
You'll need the source code if you want to work on datav itself.

### <Heading number="1">Start the backend service</Heading>
```bash
> git clone https://github.com/apm-ai/datav
> cd datav
> go build
> ./datav generate ## Required only on fresh start or new plugin added
> ./datav &
```

### <Heading number="2">Start the UI</Heading>
```bash
```bash
> cd ui
> nvm use    
> yarn install
> yarn start
```
