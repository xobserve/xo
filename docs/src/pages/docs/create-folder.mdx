很多时候，我们都需要对仪表盘进行分组，这样能够方便查看和管理，在Datav中主要是通过文件夹进行分组管理，管理员可以创建文件夹
<TipCompat>只有Global Team的管理员可以创建和管理文件夹</TipCompat>

## 创建文件夹
1. 点击左侧菜单栏，选择**+ ->文件夹**， 进入文件夹管理页面
2. 点击新建文件夹按钮
3. 输入文件夹名称test，要注意的是，名称不能重复
4. 点击提交按钮

当创建成功后，我们可以在当前的列表中看到我们新建的test文件夹


## 管理文件夹
把鼠标悬浮在点test文件夹所在的行上，会在该行的最右边浮现出一个设置按钮，点击按钮可以进入test的管理页面。

在此页面中，我们能够查看该文件夹拥有的仪表盘，也能对文件夹的属性进行设置，例如修改名称等。

## 更改仪表盘所属的文件夹
还记得之前，我们创建的文件夹属于General文件夹把？这个文件夹是datav默认的，现在我们对仪表盘所属的文件夹进行下更改。

1. 打开搜索面板，找到刚才创建的仪表盘(在General文件夹下)，点击进入仪表盘
2. 使用快捷键command + s或者点击右上方的保存按钮，打开保存面板
3. 在文件夹一栏，选择我们刚创建的test文件夹
4. 点击保存

好了，此时仪表盘已经归属于test文件夹下，可以打开搜索面板进行验证。
